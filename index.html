Bandung Coffeeshop API

Bandung Coffeeshop API is a free, simple, and lightweight API that provides Coffeeshop data in Bandung, Indonesia. Based on Open Data Kota Bandung

Getting Started

Base URL

http://coffeeshop.ihsbramn.tech/

Get all Coffeeshop Data

Endpoint : /get

Method : GET

curl --location --request GET 'coffeeshop.ihsbramn.tech/get'

Get a Specific Coffeeshop by ID

Endpoint : /get/{id}

Method : GET

curl --location --request GET 'coffeeshop.ihsbramn.tech/get/1'

Search a Coffeeshop by Name

Endpoint : /search/

Method : POST

Content Type : application/x-www-form-urlencoded

Key : name

curl --location --request POST 'coffeeshop.ihsbramn.tech/search/' \
                    --header 'Content-Type: application/x-www-form-urlencoded' \
                    --data-urlencode 'name=yumaju'
